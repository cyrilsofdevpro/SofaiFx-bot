🔐 SECURITY ARCHITECTURE (6 LAYERS)
====================================

Layer 1: USER AUTHENTICATION
  - JWT tokens (dashboard)
  - API keys (EA/bots)
  - UserContext extracts user_id

Layer 2: CREDENTIAL ENCRYPTION
  - Fernet symmetric encryption
  - Credentials encrypted in DB
  - Decrypted only when needed

Layer 3: ISOLATED MT5 CONNECTIONS
  - One session per user
  - Thread-safe with RLock
  - Separate MT5 process per user

Layer 4: SIGNAL ISOLATION
  - Signals tagged with user_id
  - Queries filtered by user_id
  - User sees only their signals

Layer 5: TRADE EXECUTION ISOLATION
  - Trades executed on user's MT5 only
  - Each trade tagged with user_id
  - Risk management per user

Layer 6: DATA ACCESS CONTROL
  - All API requires authentication
  - Database queries filtered by user_id
  - 401/403 errors for unauthorized access
